# React Task

## Basic Setup Documentation

This is a basic setup for a React project.

### Prerequisites
- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jayanthdev1/react-task.git
   cd react-task
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

### Build
To create a production build, use:
```bash
npm run build
```
